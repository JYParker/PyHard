PYHARD

업로드된 파일에서 개인정보를 자동으로 탐지하고 사용자에게 경고하는 보안 중심 웹하드

🔍 프로젝트 개요

이 프로젝트는 사용자가 업로드한 파일을 자동으로 검사하여, 주민등록번호, 전화번호, 이메일 주소 등 개인정보 포함 여부를 탐지하고, 필요 시 사용자에게 경고 이메일을 전송하는 웹하드 시스템입니다.

📁 개발환경

VSCODE

GitHub

⚙️ 기술 스택

  • Flask - Python 기반 웹 프레임워크로 회원관리 및 파일 업로드/다운로드 처리
  • MongoDB - 로그 및 회원정보, 파일 정보 저장
  • SMTP - 위험 파일 탐지 시 사용자에게 알림 메일 전송

🗂 주요 기능

  • 로그인을 통한 개인 저장소 제공
  • 파일 업로드 UI 제공
  • 정규표현식 기반 개인정보 탐지
  • MongoDB에 검사 로그 저장
  • 위험 탐지 시 이메일 경고 전송

🚀 프로젝트 실행 방법

  1. 레포지토리 클론: git clone https://github.com/JYParker/PyHard.git
  2. 필요한 패키지 설치: pip install -r requirements.txt
  3. 환경 변수 설정: util/.env 파일 생성
  4. 서버 실행: app.py
